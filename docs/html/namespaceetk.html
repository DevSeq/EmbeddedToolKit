auto etk::constrain_circular ( auto  x,
uint32  segments 
)

Used to constrain a number to within a circular range. For example,.

cout << constrain_circular(450, 360);

Output: 90 Why? Because 450 degrees is the same as 90 degrees on a map. This function can be used with radians by making segments 2*PI.
