template<typename T>
class etk::LoopRange< T >

This class is of limited usefullness by itself. It should be used with the range() template functions.

for(auto i : range(4))
cout << i << " ";
Output: 0 1 2 3
for(auto i : range(5, 9))
cout << i << " ";

Output: 5, 6, 7, 8
