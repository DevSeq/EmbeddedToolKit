Detailed Description

template<class T>
class etk::Bits< T >

Bits provides an easy way to set and read the bits of an integer individually.

etk::Bits<uint8_t> bits;
bits.set_bit(7);
bits.set_bit(1);
uint8_t v = bits.get();
auto str = bits.to_string();
cout << "v == " << str.c_str() << endl;
Template Parameters
TAn int type. For example, uint16_t which is 16 bits long.
