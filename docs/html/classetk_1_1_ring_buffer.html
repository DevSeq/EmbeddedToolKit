Detailed Description

template<class T, bool overwrite = false>
class etk::RingBuffer< T, overwrite >

RingBuffers are a great way to buffer information from data streams such as UARTs. RingBuffer does not allocate it's own memory, instead it requires a pointer to an array or memory region where it can store data.

Example.

char usart_buffer[128];
etk::RingBuffer<char> ringbuf(usart_buffer, 128);
void usart_isr_hander()
{
ringbuf.put(USART_READ_BYTE());
}
void foo()
{
while(ringbuf.available())
Usart.put(ringbuf.get());
}
Template Parameters
Ttype of object that is being buffered. For a UART this would typically be a char or uint8.
overwriteIf overriden to true, the ring buffer will write over data when it becomes full.
